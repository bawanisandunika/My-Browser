🌐 My Browser
My Browser is a lightweight, easy-to-use Android web browser designed to make your internet surfing smooth and efficient. 🚀

✨ Features
🖥️ Simple UI: Intuitive and user-friendly interface.
⚡ Fast Browsing: Quick page load times with optimized performance.
🔙🔜 Navigation Controls: Easily go back, forward, refresh, and share pages.
🔍 URL Bar: Smart URL bar with auto-suggestions and search capabilities.
🔎 Built-in Zoom: Enable or disable zoom controls for better readability.



📥 Installation
Clone the repository: https://github.com/bawanisandunika/My-Browser

🚀 Usage
Enter a URL or search term in the address bar.
Use the back, forward, refresh, and share buttons for easy navigation.
Enjoy a smooth browsing experience with integrated JavaScript and zoom controls.
